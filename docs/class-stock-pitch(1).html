PROBLEM
형성평가 없이
수업 중 이해도를 읽을 수 있다면?
"손들어 봐, 이해했어?" — 대답하는 학생은 없다. 기존 퀴즈 도구는 수업을 끊고, 반복하면 흥미가 떨어진다. 교사에겐 실시간 피드백이, 학생에겐 수업 참여의 이유가 필요하다.
👩‍🏫
교사의 문제
수업 중 학생 이해도를 실시간으로 파악할 수 없음. 카훗 같은 도구는 수업 흐름을 끊는 별도 활동. 형성평가를 따로 하기엔 시간이 부족.
🧑‍🎓
학생의 문제
수동적 청취 → 집중력 저하 → 핵심 놓침. "맞히면 점수" 구조는 틀리면 창피함만 남겨 저성취 학생이 이탈.
🔄
기존 도구의 한계
카훗, 퀴즈렛 등은 반복 사용 시 신기함 소진. 수업 흐름과 별개로 작동하며, "정답 맞히기"라는 수렴적 사고만 유도.
🛡️
개인정보 우려
기존 도구는 계정 가입 필수. 영상/음성 데이터 수집 우려. Class-Stock은 터치 인터랙션만 수집, 가입 불필요.

REPORT
교사가 반복 사용하는
진짜 이유
"재미"가 아니라 "이 데이터가 내 수업에 도움이 되니까". 정산 즉시 생성되는 리포트로, 다음 수업을 바로 설계할 수 있다.
📊
인지-확신 갭 분석
찜하기율(인지) × 매수율(확신) 이중 바 차트 + 사분면 매트릭스로 "알았는데 확신 없는 개념" vs "완전히 놓친 개념" 구분
찜 45% / 매수 23% → "ATP는 다른 방식으로 다시 설명 필요"
🔥
만장일치 / 논쟁 종목
87%가 동일 판단한 만장일치 종목 vs 매수 52% / 미매수 48%로 갈린 논쟁 종목 — 토론 소재 자동 추출
논쟁: #세포호흡 → "비교 토론 활동 권장"
🔬
학생 행동 분석
매도율(확신 번복), 추가매수율(확신 강화), 찜→매수 전환율, 매수 타이밍(즉시/마감직전) — 행동 패턴으로 이해도를 역추적
ATP 매도율 42% → "학생들이 확신을 번복한 혼란 개념"

🧠
자기결정성 이론
학생이 스스로 "이게 중요하다"를 판단. 정답 강요 없이 능동적 의사결정 → 자율성 · 유능감 · 관계성
🎰
가변비율 강화
히든 종목이 매 수업 존재. 아무도 주목 안 한 키워드가 ×3 폭등하는 서프라이즈 → "전체 수업에 주의 분산"
💔
손실 회피
10,000pt를 "이미 가진 상태"에서 시작. 잃는 아까움이 받는 기쁨의 2배 (Kahneman) → 다음 수업 동기
📈
엔다우먼트 효과
학기 내내 누적 포트폴리오. 한 번 쌓은 자산은 잃고 싶지 않음 → 중도 이탈 방지, 지속적 참여
